# Asset Bookmarks

## Git

- `.idea` はリポジトリで管理する。ルートの `.gitignore` へ追加したり、既存の設定ファイルを削除したりしない。

## UI

- 小さくドッキングしたウィンドウで多数のBookmarkを一覧できることを最優先する。
- ヘッダーと各行は最小限にし、Bookmarkは1行のコンパクト表示を基本とする。
- Bookmarkの追加はウィンドウ全面のドラッグ＆ドロップで行い、常設の追加ボタンは置かない。
- 全体のEditモードは設けず、項目ごとの動作変更・並べ替え・削除は右クリックメニューから行えるようにする。
- コンパクト表示を既定としつつ、文字・アイコン・行高をまとめて変える表示サイズ設定を用意する。
